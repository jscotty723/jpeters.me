ConnectFour Explained:


My process in developing this game:
After spending a lot of time in ReactJS, I wanted to challenge myself to build this with just VanillaJS. This was agreat project to remember JavaScrpit and also DOM manipulation.

Technologies Used:
• HTML5
• CSS3
• VanillaJS

Barriers to Development:
After being in React and working with JSX syntax, I had to shift my mindset a bit to apprach this project. The biggest issue was figuring out how to update and change the board as the game was played using DOM manipulation. Also, styling wise, I haven't worked much with HTML tables, so figuring out how to reset the CSS of one was a important step. I watched a tutorial of how someone accomplished this using JQuery in order to orient myself a bit to the process.

Refactoring:
The first refactor was to allow a user to pick the color with which their mark would render. The second refactor was to completely redo the CSS for the project so that it could handel browsers being in different sizes.
