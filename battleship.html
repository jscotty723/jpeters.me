Battleship Explained:


My process in developing this game:
This project began while in web development school as a exercise in learning ReactJS. While utilizing pair programming, we were able to put together a rudimentary version of this game over a couple of days. I then refactored the program on muy own, adding functionality and styling.

Technologies Used:
• HTML5
• CSS3
• ReactJS

Barriers to Development:
The innitial barrier to developing this was understanding ReactJS. Learning to use State, Props, the Constructor, and components set initial challenges. We also struggled to render the boxes and and control the way they displayed using the function in the render section. When I brok the project out onto my own, I refactored the program a bit, adding different difficulty levels. I also was able to learn more solidly some principles of using FlexBox to style the final app.

Refactoring:
My innital refactoring cleaned up some of the code on how the ships were generated. Next, I concentrated on the styling and user expierence that was delivered. Finally, I added the functionality of having different difficulty levels with both giving the user less torpedos, and also making more ships to find.
