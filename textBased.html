Text Based Adventure Game Explained:

My process in developing this game:

This game was developed while I was in my coding bootcamp in order to expose me to Ruby. We worked through writing the story and then coding the different snipits together so that the user could move through various versions of the story depending on their choices as the game progressed. There was some criteria involving testing and also using a shovel operator or referencing a physical shovel in the game text.

Technologies Used:

● Ruby

Barriers to Development:

This was my first time using a backend language. It felt difficult to conceptualize it well with no true "view" component.

Refactoring:

I'd love to add a typewriter effect to it to help the reader keep interest in longer portions of the sotry. This would allow the text to appear less overwhelming and more at the pace the reader is reading.
