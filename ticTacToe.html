Tic-Tac-Toe Explained:


My process in developing this game:
This was the first application I built while learning React. Learning to use state to change the styling was one of my biggest take aways. While I class we got a version functioning, but amytime the browser was changed, the board would get completely unaligned.

Technologies Used:
• HTML5
• CSS3
• ReactJS

Barriers to Development:
Not understanding how to format using flexbox was a huge issue in my first attemption. While the logic worked, the layout did not. After starting over completely with the CSS, my second attempt worked much better to support different browser sizes.

Refactoring:
The first refactor was to allow a user to pick the color with which their mark would render. The second refactor was to completely redo the CSS for the project so that it could handel browsers being in different sizes.
